在雅思写作（无论是Task 1小作文还是Task 2大作文）中，连接词（Connectors/Linking words）是评分标准中**连贯与衔接（Coherence and Cohesion）**的关键得分点。

为了方便记忆和使用，我将这些连接词按照**逻辑关系**进行了分类，并为你详细列出了它们的准确含义、使用场合以及非常重要的禁忌事项（雷区）。

---

### 一、 递进与补充关系 (Addition)
用于在已有观点的基础上，补充新的理由或信息，通常用于大作文的主体段落。

| 连接词 / 词组 | 准确含义 | 使用场合 | 禁忌事项 / 注意点 |
| :--- | :--- | :--- | :--- |
| **Furthermore** / **Moreover** | 此外，而且 | 强调后面补充的观点比前面的更重要或更深入。非常正式。 | **禁忌：** 不要在一个段落里连续使用。它们属于强逻辑词，过度使用会显得生硬。 |
| **In addition** / **Additionally** | 另外，此外 | 中规中矩的补充说明，用于引出平行的下一个论点。 | 后面通常紧跟逗号（In addition, ...）。 |
| **Besides** | 况且，除...之外 | 引出额外的理由。 | **禁忌：** 在雅思Task 2中略显口语化，建议多用 *Furthermore* 替代单用的 *Besides*。 |
| **Not only... but also...** | 不仅...而且... | 强调两个方面都成立，常用于长难句或倒装句以提升句型多样性。 | **禁忌：** 必须保证 *not only* 和 *but also* 后面的语法结构完全对称（如都是名词或都是动词）。 |

---

### 二、 对比与转折关系 (Contrast & Concession)
用于引出对立观点、反方意见或进行让步。这是展现批判性思维（Critical Thinking）的高频词汇。

| 连接词 / 词组 | 准确含义 | 使用场合 | 禁忌事项 / 注意点 |
| :--- | :--- | :--- | :--- |
| **However** | 然而 | 最通用的转折词，用于推翻前文或引出对立面。 | **禁忌：** 绝对不能像 *but* 一样直接连接两个完整的句子（除非用分号）。例如不能写 `I like apples, however she likes bananas.` 正确做法是 `..., but she...` 或 `; however, she...`。 |
| **Nevertheless / Nonetheless** | 尽管如此 | 比 However 语气更强、更正式，通常在承认了某个事实后，依然坚持某种相反的结论（让步转折）。 | 使用场景需带有强烈的“退一步讲”的意味，不要滥用作普通的“但是”。 |
| **By contrast / In contrast** | 相比之下 | 客观比较两个事物的不同点。多用于Task 1（图表题）对比数据，或Task 2对比两类人群。 | **禁忌：** 它只表示“不同”，不带有推翻对方的意味。 |
| **On the contrary** | 恰恰相反 | 用于**强烈反驳**前文的错误陈述，并提出正确的观点。 | **绝对禁忌：** 经常被误用为普通的转折！只有在前文是一个“否定句”或“谬误”时，才能用它来反驳。 |
| **Although / Even though** | 尽管，虽然 | 引导让步状语从句，用于句首或句中。 | **绝对禁忌：** 后面接了完整句子后，主句**不能**再加 *but*！（虽然...但是...在英语里只能出现一个）。 |
| **Despite / In spite of** | 尽管 | 含义同 Although，但词性是介词。 | **绝对禁忌：** 后面**绝对不能**接完整的句子，只能接名词、动名词（-ing）或名词性短语。 |

### 三、 原因与结果 (Cause & Effect)
用于分析社会现象的成因，或推导某项政策/行为带来的影响。

| 连接词 / 词组 | 准确含义 | 使用场合 | 禁忌事项 / 注意点 |
| :--- | :--- | :--- | :--- |
| **Therefore / Consequently** | 因此，结果是 | 极其正式的因果推导，常用于得出结论或陈述必然的后果。 | 通常位于句首并加逗号，或位于分号之后（; therefore,）。 |
| **As a result** | 结果 | 叙述某个事件直接导致的客观后果。 | 比较通俗易懂，Task 1 和 Task 2 都适用。 |
| **Thus / Hence** | 因而，由此 | 非常地道且高级的因果词，往往带有一点逻辑学上的推论色彩。 | **注意：** *Thus/Hence* 后面不仅可以接完整的句子，还可以直接加动词ing形式（例：*..., thus leading to...*），这是雅思高分句型。 |
| **Because of / Due to / Owing to** | 由于，因为 | 引出原因，词性为介词短语。 | **禁忌：** 后面只能接名词或动名词短语，不能接完整句子。 |
| **Stem from / Be attributed to** | 源于 / 归因于 | 作为动词短语使用，用来深度剖析深层原因。 | 属于高阶词汇表达，替代烂大街的 *The reason is...*。 |

---

### 四、 举例与例证 (Example & Illustration)
空泛的说理无法拿高分，必须用具体例子支撑。

| 连接词 / 词组 | 准确含义 | 使用场合 | 禁忌事项 / 注意点 |
| :--- | :--- | :--- | :--- |
| **For example / For instance** | 例如 | 最稳妥的举例词，位于句首或句中插入语。 | **禁忌：** 不要在一篇文章里重复使用，两者交替使用即可。 |
| **To illustrate / A prime example is** | 为了说明这一点 / 一个典型的例子是 | 开启一个相对较长的、具体的案例分析。 | 适合用来凑字数并展开长难句，显得非常有学术感。 |
| **Such as** | 比如 | 用于在一个句子内部列举具体的名词项目。 | **禁忌：** 不能放在句首！不能用 *like* 替代（*like* 在雅思写作中太口语化）。 |
| **Namely** | 即，也就是 | 对前文提到的抽象概念进行具体的点名或解释。 | 后面通常直接跟名词。 |

### 五、 顺序、分层与列举 (Sequence & Order)
用于让文章结构清晰，考官能一眼看出你的行文脉络。

| 连接词 / 词组 | 准确含义 | 使用场合 | 禁忌事项 / 注意点 |
| :--- | :--- | :--- | :--- |
| **First and foremost / Primarily** | 首要的是，首先 | 用于引出主体段最重要、最有力的第一个论点。 | 比干瘪的 *Firstly* 要高级得多。 |
| **Subsequently** | 随后，紧接着 | 多用于 Task 1 流程图（Process）中表示下一个步骤。 | 大作文（Task 2）中较少用，小作文流程图中是提分神器。 |
| **Meanwhile / In the meantime** | 与此同时 | 表示同一时间发生的另一件事，或者用于Task 1中并列图表的过渡。 | 不要和 *at the same time* 混淆语法结构。 |
| **Finally / Lastly** | 最后 | 引出最后一个论点或步骤。 | **绝对禁忌：** 千万不要用 **At last**！*At last* 带有强烈的感情色彩，表示“历经千辛万苦终于...”，在客观学术写作中非常不恰当。 |

---

### 六、 总结与结论 (Conclusion)
只用于文章的最后一段，给考官明确的收尾信号。

| 连接词 / 词组 | 准确含义 | 使用场合 | 禁忌事项 / 注意点 |
| :--- | :--- | :--- | :--- |
| **In conclusion** | 总而言之 | 最标准、最安全的结尾段开头标志。 | 没有任何禁忌，雅思官方考官范文也大量使用。 |
| **To summarize / In summary** | 概括来说 | 同上，可以作为 *In conclusion* 的完美平替。 | 同样安全可靠。 |
| **Ultimately** | 归根结底 | 带有深层总结的意味，强调最终的、最核心的结论。 | 适合在结尾段重申个人最核心的立场时使用。 |
| **By and large** | 总体而言 | 用于对某个复杂现象做一个宏观的定性。 | **禁忌：** 不要使用 *In a nutshell*, *To sum up*, 或 *All in all*，这些词对于雅思学术类（A类）写作来说**太不正式了（Too informal）**。绝不能生造 *As a conclusion*。 |

---

### 💡 雅思连接词使用的“黄金法则”

1.  **切忌“过度连接”（Over-cohesion）：** 不要每句话的开头都硬塞一个连接词。雅思考官非常反感机械化的模板（例如：Firstly... Secondly... Thirdly... Finally...）。好的衔接往往是通过**代词指代（this, these, such）**或者**同义替换（Synonyms）**隐性完成的。
2.  **标点符号是重灾区：** 像 *However*, *Therefore*, *Furthermore* 这些副词性的连接词，如果放在两句完整的话中间，前面必须是句号（.）或分号（;），后面必须有逗号（,）。
3.  **注意文体正式度（Formality）：** 放弃使用 *besides, so, and, but, kids, a lot of* 等口语词汇，全面替换为 *furthermore, therefore, additionally, however, children, a great deal of*。
